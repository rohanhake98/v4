## How to Run Locally üíª

Follow these steps to run this portfolio on your local machine.

### Prerequisites
- Node.js (v18 or higher)
- npm (comes with Node.js)

### Clone this repository

```bash
git clone https://github.com/rohanhake98/v4.git
```

### Navigate to directory

```bash
cd my-portfolio-1-main
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm start
```

Now, you can navigate to [http://localhost:3000/](http://localhost:3000/) to view the app.

### Build for Production

```bash
npm run build
```
